Comment fonctionne l'apprentissage ?

On prend les 4.28 millions de numeros et on les separe en 2 groupes :
- 70% (cours) : le modele apprend en voyant les reponses ("ce numero est fraude, celui-la est normal")
- 30% (examen) : le modele doit deviner TOUT SEUL, sans voir les reponses

Si le modele devine bien sur les 30% → il est bon ! C'est exactement comme un etudiant qui apprend en cours (70%) puis passe l'examen (30%).

Comment verifier que le test est bon ?

F1-Score : C'est le chiffre le plus important. Entre 0 et 1.
- F1 > 0.9 = Excellent (le modele est tres bon)
- F1 entre 0.7 et 0.9 = Bon (utilisable)
- F1 < 0.5 = Mauvais (le modele devine au hasard)

Matrice de Confusion : regarde les 2 chiffres importants :
- Faux Positifs (en rouge) = numeros normaux accuses par erreur → on veut le MINIMUM
- Faux Negatifs (en orange) = fraudeurs non detectes → on veut le MINIMUM

Notre resultat : F1 = 0.997 avec seulement 42 faux negatifs sur 318,000 fraudes. Le modele rate moins de 0.01% des fraudeurs.
